## State change (R1)

The world's nature changes category: from *fantasy landscape* to *running system with a debug layer* — and the protagonist's identity changes with it. He isn't just reincarnated into the world. According to the world, he might be the one holding root access. The `(?)` changes everything twice.

## Hook (R5)

Over his own trunk, in the three seconds the debug overlay renders: **`ADMIN(?)`** — half-corrupted, unresolved, and gone before he can read it twice.

## Script

**RECAP BEAT — 0:00–0:06**

The field from 001. The distant footsteps pass by without stopping — this branch is about what happens when *nothing* comes.

> **VO:** Day nine. No visitors. No threats. No events of any kind. Which means it's time for the only religion I've ever practiced: *systematic testing.*

**THE TEST LOG — 0:06–0:40**

A rapid montage. Each experiment is one shot plus one line of a growing on-screen log, deadpan:

*He strains at nothing, hard.*

```
TEST 12: think "menu" very hard............ null
```

*He strains harder. A bird lands on him, unimpressed.*

```
TEST 13: think "STATUS WINDOW" in caps..... null
TEST 14: think it in a cool anime voice.... null (bird acquired)
```

*Timelapse: he grows two twigs at precise angles.*

```
TEST 23: grow twigs at exactly 90°......... success. system response: none.
TEST 24: spell "HELLO" in root pattern..... success. readable only by moles.
TEST 25: mole response..................... ate the H
```

*Night. Stars wheel overhead.*

```
TEST 31: pray.............................. embarrassing. null.
TEST 32: threaten the sky.................. null. bird came back though.
```

> **VO:** Conclusion after nine days: this world has no interface, no console, no cheat menu. It's just… a world. Which would be fine. Except—

*Freeze on him mid-strain.*

> **VO:** —except reincarnation happened. Somebody *implemented* that. And in my experience, nobody builds a feature that big without leaving a debug flag on somewhere.

**SUNSET — 0:40–1:00**

Golden hour. The light goes long and amber. The bird flies home. Everything is soft, warm, cinematic —

— and then, for exactly three seconds, **the sun touches the horizon** and the world *flickers*.

WIREFRAME. Everything at once: the grass is a lattice of green vectors, the rocks are low-poly meshes, the distant hills are LOD billboards, the bird overhead is a rigged armature trailing a label. Floating tags snap open over every object, clean monospace on the amber sky:

```
GRASS      v2.3.1-stable
ROCK       static · no owner
STREAM     service · flow=managed
BIRD       npc_ambient · loop=daily
```

> **VO (whisper):** …There it is. There's the debug layer. Render distance: everything. Refresh: sunset. *Somebody left the flag on.*

**THE LABEL / HOOK — 1:00–1:25**

He looks down — at himself.

Over his own thin trunk, a tag renders. It stutters. Half the characters are corrupted, resolving and failing, resolving and failing:

```
▓▓P▓IN(?)
A▓M▓N(?)
ADMIN(?)
```

> **VO:** That's… that's not `npc`.

The sun slips below the horizon. The wireframes collapse, the tags fold shut, the world snaps back to soft, ordinary dusk — grass, rocks, stream, as if nothing happened.

He stands alone in the dark, one small plant in an enormous field.

> **VO:** ADMIN. Question mark. *Whose* question mark? Mine — or the system's?

Beat.

> **VO:** …Seventeen hours until the next sunset.

SMASH TO BLACK.

```
TEST 33: wait.
```
